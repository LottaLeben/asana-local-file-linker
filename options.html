Settings

⚡ How to Use

  1. Open any Asana task that contains a local file path in its description or comments
  2. Recognized paths appear highlighted in blue automatically
  3. Hold Alt (Windows/Linux) or ⌥ Option (Mac) — the cursor changes to a pointer over paths
  4. Alt+Click / Option+Click the path — the file opens in your default app

Supported path formats: /Users/… · /Volumes/… · /home/… · C:\… · \\server\… · custom URL schemes. Add more via Custom Path Patterns below.

🚫 Blocked file type? If a file is blocked for security reasons, you'll see an orange warning. Add the extension to the Allowed File Types list below to override.

📋 Clipboard fallback: If the native host is not installed, the path is copied to your clipboard instead. Use ⌘+Shift+G (Mac Finder) or Win+R (Windows) to navigate to it.
